# Manifesto Pyenv

## Versoes Instaladas
- 3.11.8: compativel com projetos legado, instalar deps via pip-tools quando necessario.

## Procedimentos
- Criar nova versao: pyenv install <versao>.
- Definir versao local: pyenv local <versao>.
- Gerenciar ferramentas globais com pipx (ex: pipx install cookiecutter).
